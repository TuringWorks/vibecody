# VibeUI Example

This is an example file to test the VibeUI code editor!

## Features

- **Monaco Editor**: The same editor that powers VSCode
- **Rust Backend**: Fast and efficient text processing
- **AI Integration**: Code completion and chat with multiple providers

## Try it out

1. Open a folder using the "Open Folder" button
2. Click on any file in the sidebar to open it
3. Start editing!

## Code Example

```rust
fn main() {
    println!("Hello from VibeUI!");
}
```

```typescript
const greet = (name: string) => {
  console.log(`Hello, ${name}!`);
};
```

Enjoy coding with VibeUI! 🚀
