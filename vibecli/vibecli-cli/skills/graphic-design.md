# Graphic Design

When working with graphic design:

1. Apply typography fundamentals by establishing clear visual hierarchy (use size, weight, and spacing to guide reading order), pair typefaces with contrasting characteristics (a serif heading with a sans-serif body), maintain consistent line height (1.4-1.6x for body text), set comfortable line lengths (45-75 characters), use proper typographic characters (em dashes, curly quotes, ligatures), and never use more than three typefaces in a single project.
2. Use color theory intentionally by selecting palettes based on established harmonies (complementary for high contrast, analogous for cohesion, triadic for vibrancy), ensuring sufficient contrast ratios for accessibility (4.5:1 minimum for body text per WCAG), defining a primary, secondary, and accent color with neutrals, accounting for color blindness (avoid relying on red/green distinction alone), and testing colors across screens and print proofs.
3. Build layouts on grid systems by defining consistent margins and gutters, using column grids for text-heavy documents (6, 9, or 12 columns for flexibility), applying modular grids for complex layouts with both horizontal and vertical divisions, aligning all elements to the grid while allowing intentional breaks for emphasis, and maintaining consistent spacing with a baseline grid for text alignment.
4. Understand vector versus raster workflows: use vector (Illustrator) for logos, icons, illustrations, and anything that needs to scale infinitely; use raster (Photoshop) for photo manipulation and complex compositing; combine both in layout tools (InDesign) for final production; and export vectors as SVG for web, EPS or AI for print production, and PDF for universal distribution.
5. Master the Illustrator Pen tool by clicking for corner points and click-dragging for smooth curves, holding Alt/Option to break handles for cusp points, using as few anchor points as possible for smooth paths, practicing with tracing exercises to build muscle memory, and leveraging the Curvature tool for faster organic shapes; always close paths for filled shapes.
6. Create InDesign multi-page documents by setting up master pages with consistent headers, footers, and page numbers, using paragraph and character styles for every text element (never override manually), defining object styles for repeated design elements, using text threading for flowing copy across frames and pages, and building a table of contents from paragraph styles for long documents.
7. Prepare files for print production by setting up documents in CMYK color mode from the start, adding 3mm (0.125 inch) bleeds on all sides where design extends to the edge, converting all fonts to outlines or packaging with the document, using spot colors (Pantone) for brand-critical colors, setting image resolution to 300 PPI at final print size, and creating a preflight profile to catch errors before sending to press.
8. Follow a logo design process that begins with a thorough brief (audience, values, competitors), progresses through research and moodboarding, sketches 20-30 concepts by hand, refines 3-5 directions digitally in black and white first, tests the mark at small sizes and in single color, presents options with mockups in context, and delivers final files in all required formats (vector, raster, reversed, stacked, horizontal).
9. Create comprehensive brand guidelines by documenting logo usage (clear space, minimum size, prohibited modifications), defining primary and secondary color palettes with exact values (Pantone, CMYK, RGB, HEX), specifying typography (primary and secondary typefaces with usage rules), establishing photography and illustration style, defining iconography, providing layout templates, and including real-world application examples (business cards, signage, digital).
10. Prepare files correctly for print versus digital delivery: print requires CMYK color, 300 PPI images, outlined fonts, bleed marks, and PDF/X-1a or PDF/X-4 export; digital requires RGB color, 72-150 PPI images, embedded or web-safe fonts, and optimized file sizes; social media assets need platform-specific dimensions; and always maintain an editable source file alongside flattened deliverables.
11. Design icon and illustration systems with consistent stroke weights, corner radii, and grid sizes across the entire set; use a standardized artboard size (24x24, 32x32 pixels for UI icons); maintain optical balance (a circle must be slightly larger than a square to appear the same size); test at final display size for clarity; and export with appropriate formats (SVG for web, PDF for print, PNG at multiple densities for mobile).
12. Conduct design critique and feedback by presenting work with context (the brief, target audience, constraints), inviting specific feedback rather than general opinions, separating subjective preference from objective effectiveness, iterating based on feedback priorities (function over style), using annotations to explain design decisions, and maintaining version history to track the evolution of the design.
