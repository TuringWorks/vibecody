# Photography & Image Editing

When working with photography and image editing:

1. Master the exposure triangle by understanding that aperture controls depth of field (f/1.8 for shallow bokeh, f/8-f/11 for sharpness across the frame, f/16+ for deep landscapes), shutter speed freezes or blurs motion (1/500+ for sports, 1/60 for handheld minimum, long exposures for light trails), and ISO amplifies signal at the cost of noise (keep as low as possible, modern cameras are clean to ISO 3200-6400).
2. Apply composition rules deliberately: use the rule of thirds to place subjects at intersection points, employ leading lines to guide the viewer's eye, create depth with foreground interest, use framing elements (doorways, branches) to draw attention, apply negative space for minimalist impact, and break rules intentionally once you understand them.
3. Shoot RAW for maximum editing flexibility (14-bit color depth, recoverable highlights and shadows, non-destructive white balance adjustment) and use JPEG only for rapid delivery where post-processing is unnecessary; set camera to RAW+JPEG if you need quick previews alongside archival files.
4. Organize Lightroom catalogs by importing with a consistent folder structure (Year/Month/Event), applying keywords and metadata on import, using collections and smart collections for project-based organization, rating images with stars (1=keep, 3=portfolio candidate, 5=best), using color labels for workflow status (red=needs editing, green=complete), and backing up the catalog file separately from image files.
5. Edit non-destructively in Photoshop by always working with layers (never flatten until final export), using adjustment layers for color and tonal changes, applying masks to control where edits appear, converting layers to smart objects before applying filters (enabling re-editable parameters), and saving working files as PSD or PSB to preserve layer structure.
6. Correct color and white balance by first setting a neutral point using the eyedropper on a known gray area, adjusting temperature (warm/cool) and tint (green/magenta) to match the scene, using HSL sliders for selective color control, applying split toning for creative color grading (warm highlights, cool shadows), and calibrating your monitor with a hardware colorimeter for accurate representation.
7. Batch process efficiently by creating Lightroom develop presets for consistent looks across a series, using Photoshop actions to automate repetitive tasks (resize, watermark, sharpen), leveraging Lightroom's sync settings to apply adjustments across multiple images, and setting up export presets for different delivery requirements (web, print, social media).
8. Create HDR images by bracketing exposures (typically 3-5 shots at 2 EV intervals), merging in Lightroom or dedicated HDR software, tone mapping to maintain natural appearance while preserving detail in highlights and shadows, and using a tripod for alignment; create panoramas by overlapping frames 30-40% and using Lightroom's photo merge with cylindrical or spherical projection.
9. Set up product photography with a seamless white background (use a light tent or sweep), employ two to three lights (key light at 45 degrees, fill to soften shadows, backlight for edge separation), shoot tethered to a computer for immediate review, maintain consistent camera settings across all products in a series, and use a tripod with a bubble level for uniform framing.
10. Approach portrait retouching ethically by preserving skin texture (avoid excessive smoothing), using frequency separation for targeted skin work, dodging and burning to enhance natural light and shadow, whitening teeth and eyes subtly, removing temporary blemishes while keeping permanent features, and communicating with subjects about the extent of retouching.
11. Manage files and backups with a 3-2-1 strategy (three copies, two different media types, one offsite), use meaningful file naming (Date_Subject_Sequence), maintain a searchable keyword database, cull ruthlessly before investing editing time, archive completed projects to external storage, and verify backup integrity periodically.
12. Optimize images for web delivery by exporting at 72 PPI (pixels per inch is irrelevant for screens, pixel dimensions matter), using sRGB color space for web display, choosing JPEG at 70-85% quality for photographs, PNG for graphics with transparency, WebP for modern browsers (30% smaller than JPEG at equivalent quality), and sizing to 2x the display dimensions for retina screens while keeping file sizes under 200KB for page performance.
