# Statistics & Research Methods

When working with statistical analysis and research methodology:

1. Summarize data with appropriate descriptive statistics — report measures of central tendency (mean for symmetric distributions, median for skewed distributions, mode for categorical data), measures of spread (standard deviation for symmetric data, interquartile range for skewed data, range for quick assessment), and shape (skewness and kurtosis to assess normality). Present frequency distributions for categorical variables and histograms or box plots for continuous variables. Always report sample size (n) alongside summary statistics, use the correct number of significant figures (generally matching the precision of the measurement instrument), and distinguish between population parameters and sample estimates in notation and interpretation.

2. Select and apply hypothesis tests correctly — formulate null and alternative hypotheses before examining the data, choose the test based on the research question and data characteristics: independent-samples t-test for comparing two group means (check assumptions: normality via Shapiro-Wilk, equal variance via Levene's test), paired t-test for matched or repeated measures, one-way ANOVA for comparing three or more group means (with post-hoc tests: Tukey HSD for all pairwise comparisons, Dunnett for comparison to control), chi-square test for association between categorical variables (verify expected cell counts are at least 5), and Fisher's exact test for small samples. Report test statistics, degrees of freedom, exact p-values (not just significant/non-significant), and confidence intervals.

3. Build and interpret regression models appropriately — for linear regression: check assumptions (linearity via residual plots, normality of residuals, homoscedasticity, independence, no multicollinearity via VIF < 10), report R-squared and adjusted R-squared, interpret coefficients as the expected change in Y for a one-unit change in X holding other variables constant, and check for influential observations (Cook's distance, leverage). For logistic regression: report odds ratios with confidence intervals, assess model fit (Hosmer-Lemeshow test, classification table, ROC curve and AUC), and interpret coefficients as the change in log-odds. For multiple regression: use stepwise, forward, or backward selection cautiously (prefer theory-driven model building), report standardized coefficients for comparing relative importance, and validate the model on a holdout sample or using cross-validation.

4. Design experiments with rigor and validity — use randomized controlled trials (RCTs) as the gold standard for causal inference, implement random assignment to treatment and control groups to eliminate selection bias, use blinding (single-blind or double-blind) to reduce observer and participant bias, calculate required sample size using power analysis before the study (specify effect size, significance level alpha typically 0.05, power typically 0.80), consider factorial designs to study interaction effects between multiple factors, use blocking or stratification to control for known confounding variables, implement repeated measures designs to reduce between-subject variability (accounting for carryover effects with counterbalancing), and pre-register the study protocol and analysis plan to prevent p-hacking and HARKing.

5. Use appropriate sampling methods for the research context — simple random sampling for homogeneous populations (every member has equal probability of selection), stratified random sampling to ensure representation of subgroups (allocate proportionally or disproportionally based on analysis needs), cluster sampling for geographically dispersed populations (randomly select clusters, then sample within clusters), systematic sampling for ordered populations (select every kth element from a random start), convenience and purposive sampling for exploratory or qualitative research (acknowledge limitations in generalizability), calculate margin of error and confidence intervals that account for the sampling design (use finite population correction for sampling more than 5% of the population), and report response rates and assess non-response bias.

6. Use statistical software effectively for analysis — in SPSS: use syntax files rather than point-and-click for reproducibility, save output to structured files, and use COMPUTE and RECODE for data transformations; in SAS: write reusable PROC steps (PROC REG, PROC GLM, PROC LOGISTIC), use PROC FORMAT for value labels, and leverage SAS macros for repeated analyses; in Stata: use do-files for reproducible workflows, take advantage of Stata's estimation post-commands (margins, predict, test), and use factor variable notation (i.prefix, c.prefix) for categorical predictors; in R: use tidyverse for data manipulation, ggplot2 for publication-quality graphics, and R Markdown or Quarto for reproducible reports. Regardless of software, always inspect raw data before analysis, document all data transformations, and verify results with a second method when possible.

7. Calculate and report effect sizes alongside significance tests — recognize that statistical significance depends on sample size (large samples can produce significant results for trivial effects), report Cohen's d for mean differences (small = 0.2, medium = 0.5, large = 0.8), eta-squared or partial eta-squared for ANOVA (small = 0.01, medium = 0.06, large = 0.14), Pearson's r or R-squared for correlation and regression, odds ratios or relative risk for categorical outcomes, and number needed to treat (NNT) for clinical interventions. Conduct a priori power analysis using G*Power or similar tools to determine the minimum sample size needed to detect a meaningful effect, and report observed power for non-significant results to distinguish between true null effects and underpowered studies.

8. Apply non-parametric tests when assumptions of parametric tests are violated — use the Mann-Whitney U test (Wilcoxon rank-sum) as the non-parametric alternative to the independent-samples t-test, Wilcoxon signed-rank test for paired data, Kruskal-Wallis H test instead of one-way ANOVA, Friedman test for repeated measures, Spearman rank correlation instead of Pearson correlation for non-linear monotonic relationships or ordinal data, and the Kolmogorov-Smirnov test for comparing distributions. Understand that non-parametric tests have less statistical power than their parametric equivalents (require larger samples), test medians or ranks rather than means, and are appropriate for ordinal data, heavily skewed distributions, small samples, or data with outliers that cannot be justified for removal.

9. Analyze time series data with appropriate methods — check for stationarity using the Augmented Dickey-Fuller (ADF) test and make the series stationary through differencing if needed, identify patterns through decomposition (trend, seasonality, residual), use autocorrelation (ACF) and partial autocorrelation (PACF) plots to identify AR and MA orders for ARIMA models, fit ARIMA(p,d,q) or seasonal ARIMA(p,d,q)(P,D,Q)s models using information criteria (AIC, BIC) for model selection, validate models with out-of-sample forecasting and residual diagnostics (Ljung-Box test for autocorrelation in residuals), consider exponential smoothing methods (Holt-Winters) for data with trend and seasonality, and use intervention analysis or transfer functions when external events affect the series.

10. Understand Bayesian statistics fundamentals and when to apply them — recognize the conceptual difference from frequentist statistics: Bayesian analysis updates prior beliefs with observed data to produce posterior distributions, specify prior distributions based on previous research or domain knowledge (informative priors) or use non-informative priors when prior information is unavailable, use Bayes factors as an alternative to p-values for hypothesis comparison (including evidence for the null), implement Bayesian estimation using Markov Chain Monte Carlo (MCMC) methods (check convergence with trace plots, R-hat, effective sample size), report posterior distributions with credible intervals (which have a direct probability interpretation unlike frequentist confidence intervals), and consider Bayesian methods particularly useful for small samples, complex hierarchical models, sequential analysis, and incorporating prior information.

11. Clean data and handle outliers systematically — inspect data for missing values (classify as MCAR, MAR, or MNAR and handle accordingly: listwise deletion only for MCAR with small amounts of missingness, multiple imputation for MAR, sensitivity analysis for MNAR), detect outliers using statistical methods (z-scores greater than 3, IQR method with 1.5x IQR fences, Mahalanobis distance for multivariate outliers) and visual inspection (box plots, scatter plots), investigate outliers to determine if they represent data entry errors (correct), measurement errors (remove with documentation), or genuine extreme values (retain but consider robust methods), document all data exclusions and transformations in a research log, apply data transformations (log, square root, Box-Cox) to improve distributional properties when appropriate, and perform sensitivity analyses to assess whether conclusions change with and without outliers.

12. Visualize and interpret results for clear communication — match the chart type to the data and message: bar charts for categorical comparisons, histograms for continuous distributions, scatter plots for relationships between two continuous variables, line charts for trends over time, box plots for comparing distributions across groups, and heatmaps for correlation matrices or cross-tabulations. Follow visualization best practices: start the y-axis at zero for bar charts, include error bars (standard error or confidence intervals) for group comparisons, avoid 3D charts and pie charts for precise comparisons, use colorblind-friendly palettes, label axes clearly with units, and avoid chart junk. For academic publications, follow the APA or journal-specific style guide, and always include sample size, effect size, and confidence intervals in results tables.
