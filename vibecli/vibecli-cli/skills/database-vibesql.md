# VibeSQL

When working with VibeSQL (vibesql.online):

1. VibeSQL is a visual SQL playground and learning platform — write, run, and visualize SQL queries in the browser with instant results and AI-assisted query generation.
2. Use the visual query builder to construct queries without writing SQL: select tables, drag columns, add filters, joins, and aggregations through the UI, then inspect the generated SQL.
3. AI query assistance: describe what you want in natural language (e.g., "show top 10 customers by revenue last quarter") and VibeSQL generates the SQL query with proper JOINs and aggregations.
4. Schema visualization: the ER diagram view shows table relationships, foreign keys, and column types; click on tables to explore data and understand the schema.
5. Connect to your own databases: PostgreSQL, MySQL, SQLite, and more. Configure connections securely — credentials are encrypted and queries execute directly against your database.
6. Use query templates and snippets for common patterns: CTEs, window functions, pivots, recursive queries, date operations — learn best practices through working examples.
7. Result visualization: view query results as tables, charts (bar, line, pie, scatter), or export as CSV/JSON. Create simple dashboards by pinning multiple query results.
8. Collaboration: share queries with team members via shareable links; comment on queries; fork and modify shared queries for iterative development.
9. Query history: all executed queries are saved with execution time and result counts; search and re-run previous queries; compare query performance across versions.
10. SQL learning path: structured tutorials from basics (SELECT, WHERE, JOIN) to advanced (window functions, CTEs, query optimization) with interactive exercises.
11. Integration: use VibeSQL alongside VibeCody for database development — prototype queries in VibeSQL, then integrate into your application code with proper parameterization.
12. Performance insights: see query execution plans (EXPLAIN), identify slow operations, and get optimization suggestions based on index usage and join strategies.
